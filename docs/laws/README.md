# Laws

This folder contains a minimal, structured legal reference set for the OSBB Platform repository.

It is intended to keep:
- summaries of laws and regulations
- documents grouped by domain
- references to official legal sources

## Structure

- `core/` -> main OSBB laws and foundational regulations
- `accounting/` -> financial and accounting rules relevant to OSBB operations
- `references/` -> aggregated official links and cross-reference lists
- `raw/` -> original texts, archived copies, or downloaded source materials if needed later

The goal is to keep legal context close to the product domain while staying simple, readable, and easy to extend.
